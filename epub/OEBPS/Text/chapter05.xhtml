第 5 章：Excel 自動化

這章要解決什麼問題

很多職場上的重複工作都圍繞在 Excel。你可能每天都要開啟表格、篩選資料、統計數字，再另存成新檔。這些工作如果每次都手動做，很容易花時間，也容易出錯。

這一章要幫你建立的能力是：

  • 把 Excel 讀進 Python
  • 根據條件篩選資料
  • 輸出成新的 Excel
  • 慢慢建立資料檢查的習慣

情境案例

假設你每週都要整理訂單資料，流程可能像這樣：

  1. 打開 orders.xlsx
  2. 找出金額大於 5000 的資料
  3. 把符合條件的資料另存成新檔
  4. 再確認有沒有篩錯

如果每週都要重複一次，這就是很適合交給 Python 的工作。

概念說明

很多重複工作都圍繞在 Excel。只要你能把 Excel 讀進 Python，就能開始做篩選、統計、轉換與輸出。pandas 可以把 Excel 資料表看成一個可操作的表格物件。

學這一章時，請先記住三件事：

  • 欄位名稱要先確認
  • 篩選條件要寫清楚
  • 輸出前最好先檢查資料筆數或前幾列

這會比只記 pandas 語法更重要。

人工流程、Python 流程、AI 協作流程

人工流程

  • 打開 Excel
  • 找欄位
  • 設定篩選條件
  • 複製結果到新檔
  • 存檔並檢查

Python 流程

  • 讀取 Excel
  • 根據條件篩選
  • 視需要保留指定欄位
  • 輸出成新檔
  • 印出處理結果

AI 協作流程

  • 根據你的欄位名稱改寫程式
  • 幫你加入分組統計或排序
  • 解釋每一行 pandas 在做什麼
  • 提醒你哪些地方可能因為欄位或資料型別出錯

程式範例

import pandas as pd

df = pd.read_excel("sales.xlsx")
high_sales = df[df["金額"] >= 10000]
high_sales.to_excel("high_sales.xlsx", index=False)

print("已輸出 high_sales.xlsx")

範例解讀

  • pd.read_excel() 讀取 Excel 資料。
  • df["金額"] >= 10000 是篩選條件。
  • df[...] 會保留符合條件的資料列。
  • to_excel() 把結果輸出成新檔。
  • index=False 表示不要把索引欄另外寫進 Excel。

輸入與輸出長什麼樣

輸入

  • 一個 Excel 檔案
  • 欄位名稱，例如 客戶、地區、金額、日期
  • 一個篩選條件

輸出

  • 一份整理過的新 Excel
  • 螢幕上的完成提示
  • 之後可延伸成摘要數字或統計表

常見錯誤與排查

  • 欄位名稱寫錯。 例如 Excel 寫的是 總金額，程式卻寫成 金額，就會出錯。
  • 金額欄位不是數字。 如果 Excel 裡的內容混有文字或空值，篩選前可能要先清理資料。
  • 輸出檔案有了，但內容不對。 建議先印出資料筆數，或先看前幾列資料，再確認篩選結果。
  • 一次做太多事。 初學者先把「讀取 + 篩選 + 輸出」做好，再慢慢加排序、分組與摘要。

AI 任務

請 AI 根據你的欄位名稱、條件與輸出需求，改寫出對應的 pandas 程式，並提醒你要先檢查哪些欄位。

AI Prompt 範例

你是一位 Python Excel 自動化助教。
我有一個 Excel 檔案 orders.xlsx，
欄位包含客戶、地區、金額、日期。

請用 pandas 寫一個適合初學者的程式，
篩選出金額大於 5000 的資料並另存新檔。

請再補充：
- 哪些欄位要先確認
- 哪裡最容易出錯
- 每段程式在做什麼

練習題

基礎題

  1. 把條件改成金額大於等於 3000。
  2. 印出資料筆數。

改寫題

  1. 只保留 客戶 與 金額 兩欄。
  2. 把輸出檔名改成 filtered_sales.xlsx。

實作題

  1. 讀取一份 Excel，篩選出指定條件的資料並另存新檔。
  2. 執行後打開輸出的 Excel，確認結果是否正確。

挑戰題

讀取一份 Excel 後依地區分組統計總金額，再輸出成新的 Excel，並額外印出銷售最高的地區。

小結

Excel 自動化最重要的不是背 API，而是先知道欄位、條件與你要的輸出，再讓 AI 協助補齊細節。當你能穩定處理 Excel，很多職場任務就已經能開始省下不少時間。
