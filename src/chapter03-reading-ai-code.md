# 第 3 章：看懂 AI 程式

## 概念說明
AI 可以很快產生程式，但真正重要的是你能不能看懂。閱讀 AI 程式最穩的方式，是先找出三件事：

1. 輸入資料是什麼
2. 處理邏輯在哪裡
3. 輸出結果去哪裡

不要一口氣讀完整個檔案，而是分段閱讀。

## 程式範例
```python
numbers = [5, 12, 18, 3]
result = []

for number in numbers:
    if number >= 10:
        result.append(number)

print(result)
```

### 範例解讀
- `numbers` 是輸入
- `result` 是輸出容器
- `for` 是重複處理
- `if` 是條件判斷

## AI 任務
請 AI 將一段你看不懂的程式拆成 4 個小段，每段都用白話解釋。

## AI Prompt 範例
```text
請把下面這段 Python 程式拆成 4 個小區塊說明。
每個區塊都要回答：
1. 這段在做什麼
2. 為什麼需要它
3. 初學者最容易卡在哪裡
```

## 練習題
1. 找出範例中的輸入、條件與輸出。
2. 把條件改成 `number >= 15`，預測結果。
3. 在迴圈中加入一行，印出目前正在檢查的數字。

## 挑戰題
請讓 AI 幫你產生一段 10 行內的檔案處理程式，然後自己分段加上註解解釋。

## 小結
真正的能力不是複製貼上，而是知道程式在做什麼。能看懂 AI 程式，才算真的會使用 AI。
