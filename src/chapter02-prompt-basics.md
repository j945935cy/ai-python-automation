# 第 2 章：Prompt 基礎

## 概念說明
Prompt 是你交給 AI 的工作說明書。Prompt 越清楚，AI 產出的程式越接近你要的結果。對初學者最有幫助的 Prompt，通常包含四個元素：

- 角色：請 AI 扮演什麼身份
- 任務：要完成什麼工作
- 限制：只能用哪些工具、程式要多簡單
- 輸出格式：希望回覆有哪些內容

## 程式範例
```python
prompt = """
你是一位 Python 初學者助教。
請示範如何列出資料夾中的所有檔案。
只能使用標準函式庫，並逐行解釋程式。
"""

print(prompt)
```

### 範例解讀
- 多行字串適合儲存 Prompt
- Prompt 本身也是教材內容，可以反覆修改與重用
- 清楚描述限制，能有效降低 AI 產生過度複雜程式的機率

## AI 任務
把你自己寫的一段模糊需求交給 AI，要求它改寫成更適合初學者的高品質 Prompt。

## AI Prompt 範例
```text
請把下面這句模糊請求改寫成適合初學者的 Prompt：
「幫我整理檔案」

請補上角色、任務、限制與輸出格式。
```

## 練習題
1. 寫一個 Prompt，要求 AI 幫你列出資料夾中的所有 PDF。
2. 寫一個 Prompt，要求 AI 用 `pandas` 讀取 Excel。
3. 寫一個 Prompt，要求 AI 解釋錯誤訊息。

## 挑戰題
寫一個完整 Prompt，要求 AI 產生「下載網頁後存成文字檔」的範例程式，並附上逐段說明。

## 小結
會寫 Prompt，不是因為你很會下命令，而是因為你知道自己真正要什麼。需求越明確，AI 的幫助越穩定。
