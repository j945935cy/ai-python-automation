# 第 9 章：報表自動化

## 概念說明
報表自動化是把前面學過的資料讀取、整理、統計與輸出整合在一起。典型流程是：

1. 讀取原始資料
2. 篩選或清理
3. 計算指標
4. 輸出報表與摘要

## 程式範例
```python
import pandas as pd

df = pd.read_excel("orders.xlsx")
summary = df.groupby("地區")["金額"].sum().reset_index()
summary.to_excel("region_summary.xlsx", index=False)

print(summary)
```

### 範例解讀
- `groupby("地區")` 依地區分組
- `sum()` 計算每組總金額
- `reset_index()` 把結果整理成一般表格

## AI 任務
請 AI 根據你的資料欄位，幫你決定哪些指標適合做成主管一眼能看懂的摘要。

## AI Prompt 範例
```text
我有一份訂單資料，欄位包含地區、商品、數量、金額、日期。
請幫我設計一個初學者可理解的報表自動化流程，
並用 pandas 寫出示範程式，輸出每個地區的總金額。
```

## 練習題
1. 改成依商品分組統計。
2. 增加一行，印出總訂單數。
3. 將輸出檔名改成今天日期開頭。

## 挑戰題
同時輸出摘要 Excel 與一段文字說明，例如「本月北區銷售最高」。

## 小結
報表自動化是很好的整合練習，因為它會逼你把資料、邏輯與輸出一次想清楚。
