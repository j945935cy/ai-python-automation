# 第 9 章：報表自動化

## 這章要解決什麼問題
前面學過的檔案處理、Excel 讀寫、資料整理，其實很常一起出現在同一個工作裡。最典型的例子，就是報表自動化。

這一章要幫你做的，不只是把 Excel 讀進來而已，而是學會把資料整理成「別人看得懂」的結果。因為報表真正的目的，不是把表格做完，而是把資訊傳達清楚。

## 情境案例
假設你每週都要整理訂單資料給主管，手動流程通常是：

1. 打開原始 Excel
2. 看有哪些欄位
3. 計算各地區總金額
4. 找出最高銷售區域
5. 另存成新報表
6. 再寫一句摘要說明

如果這些事每週都重複一次，就非常適合做成自動化流程。

## 概念說明
報表自動化是把前面學過的資料讀取、整理、統計與輸出整合在一起。典型流程是：

1. 讀取原始資料
2. 篩選或清理
3. 計算指標
4. 輸出報表與摘要

學習報表自動化時，建議先想清楚三件事：

- 你手上有哪些欄位
- 你想回答什麼問題
- 最後要輸出給誰看

這樣寫出來的程式才不會只是「有跑完」，而是真的有用。

## 人工流程、Python 流程、AI 協作流程

### 人工流程
- 開 Excel
- 用篩選或樞紐分析整理資料
- 複製結果到新檔
- 手動寫摘要文字

### Python 流程
- 讀取原始資料
- 分組統計
- 產出新報表
- 自動印出或寫出摘要

### AI 協作流程
- 幫你設計報表需求
- 根據欄位產生 `pandas` 範例
- 解釋每一步的統計邏輯
- 協助你把表格結果翻成白話摘要

## 程式範例
```python
import pandas as pd

df = pd.read_excel("orders.xlsx")
summary = df.groupby("地區", as_index=False)["金額"].sum()
summary = summary.sort_values("金額", ascending=False)

summary.to_excel("region_summary.xlsx", index=False)

top_region = summary.iloc[0]
message = f"本期銷售最高地區為 {top_region['地區']}，總金額為 {top_region['金額']}。"

print(summary)
print(message)
```

### 範例解讀
- `pd.read_excel()` 讀取原始資料。
- `groupby("地區")["金額"].sum()` 依地區加總金額。
- `sort_values()` 讓金額高的排在前面。
- `to_excel()` 把摘要表輸出成新檔。
- `summary.iloc[0]` 取出第一名地區。
- `message` 把統計結果轉成主管較容易閱讀的文字。

## 輸入與輸出長什麼樣

### 輸入
- 原始 Excel 檔
- 欄位名稱，例如 `地區`、`商品`、`數量`、`金額`
- 想計算的指標，例如總金額、平均金額、筆數

### 輸出
- 整理過的報表 Excel
- 螢幕上的摘要結果
- 可貼到訊息或 Email 的中文說明

## 常見錯誤與排查

- 欄位名稱寫錯。
  程式裡寫的欄位名稱要和 Excel 完全一致。

- `金額` 欄位不是數字。
  如果 Excel 內容混有文字，統計前可能要先清理資料。

- 只輸出表格，沒有摘要。
  主管通常不只要表格，也需要一句快速看懂的結論。

- 沒有先確認資料筆數。
  在做統計前，先看資料是否成功讀取，是很好的習慣。

## AI 任務
請 AI 根據你的資料欄位，幫你決定哪些指標適合做成主管一眼能看懂的摘要，並補上一段文字結論。

## AI Prompt 範例
```text
你是一位 Python 報表自動化助教。
我有一份訂單資料，欄位包含地區、商品、數量、金額、日期。

請幫我：
1. 設計一個初學者可理解的報表自動化流程
2. 用 pandas 寫出示範程式
3. 輸出每個地區的總金額
4. 再產生一句主管看得懂的中文摘要

請保持白話，並逐段說明程式用途。
```

## 練習題

### 基礎題
1. 改成依商品分組統計。
2. 增加一行，印出總訂單數。

### 改寫題
1. 將輸出檔名改成今天日期開頭。
2. 增加一段摘要文字，例如「本月北區銷售最高」。

### 實作題
1. 同時輸出摘要 Excel 與一段文字說明。
2. 執行後確認 Excel 內容與文字摘要是否一致。

## 小結
報表自動化是很好的整合練習，因為它會逼你把資料、邏輯與輸出一次想清楚。真正有價值的報表，不只是算出數字，而是讓別人一眼就能看懂重點。
