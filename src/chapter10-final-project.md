## 先拆成小流程
在寫程式前，先把專案拆小：

1. 確認原始檔案是否存在
2. 讀取 Excel
3. 檢查必要欄位是否存在
4. 依地區統計總金額
5. 輸出成新的 Excel
6. 找出金額最高的地區
7. 產生摘要文字
8. 把摘要寫入文字檔

這樣做的好處是，如果程式出錯，你比較容易知道卡在哪一段。

## 程式範例
```python
from pathlib import Path

import pandas as pd

source_file = Path("weekly_orders.xlsx")

if not source_file.exists():
    print("找不到 weekly_orders.xlsx，請先確認檔案是否放在同一個資料夾。")
else:
    df = pd.read_excel(source_file)

    required_columns = {"地區", "金額"}
    if not required_columns.issubset(df.columns):
        print("缺少必要欄位，請確認 Excel 是否包含 地區 和 金額。")
    else:
        summary = df.groupby("地區", as_index=False)["金額"].sum()
        summary = summary.sort_values("金額", ascending=False)

        summary.to_excel("weekly_summary.xlsx", index=False)

        top_region = summary.iloc[0]
        message = (
            f"本週銷售最高地區為 {top_region['地區']}，"
            f"總金額為 {top_region['金額']}。"
        )

        Path("summary.txt").write_text(message, encoding="utf-8")

        print("已輸出 weekly_summary.xlsx")
        print("已輸出 summary.txt")
        print(message)
```

### 範例解讀
- `Path("weekly_orders.xlsx")` 用來表示原始檔案路徑。
- `exists()` 先檢查檔案存不存在，避免一開始就報錯。
- `pd.read_excel()` 負責讀取 Excel。
- `required_columns` 用來檢查欄位是否齊全。
- `groupby("地區")["金額"].sum()` 會把每個地區的金額加總。
- `sort_values()` 讓最高金額排在前面，方便直接找到第一名。
- `to_excel()` 把統計結果輸出成新檔案。
- `write_text()` 把摘要文字寫進文字檔。

這段程式的重點不是炫技，而是讓你看見一個完整的小型工作流程。

## 常見錯誤與排查

- 檔案不存在。
  先確認 Excel 是否和程式放在同一個資料夾。

- 欄位名稱不一致。
  例如 Excel 寫成「區域」不是「地區」，這時要先改欄位名稱或修改程式。

- `金額` 不是數字。
  如果 Excel 裡混入文字，統計前要先清理資料。

- 摘要檔有輸出，但內容不合理。
  先開啟 `weekly_summary.xlsx`，確認排序和加總是否正確。
